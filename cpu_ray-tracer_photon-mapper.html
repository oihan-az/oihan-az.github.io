CPU Ray Tracer & Photon Mapper

CPU Ray Tracer & Photon Mapper

This project is a physically based offline renderer developed entirely in C++ and executed on the CPU. It allows the user to choose between classic Monte Carlo ray tracing and a solution based on Photon Mapping.

The renderer focuses on correctness, flexibility, and extensibility, supporting realistic lighting, soft shadows, caustics, and complex material interactions while maintaining a clean and modular architecture.
