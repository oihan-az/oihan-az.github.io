Infinite Ocean Surface

Infinite Ocean Surface

This project is a real-time ocean rendering system that focuses on achieving an infinite, dynamic ocean surface while maintaining interactive performance.

By combining adaptive mesh resolution with procedural wave synthesis, the system achieves a highly scalable and visually convincing infinite ocean simulation suitable for real-time applications.

Ocean mesh - Projected grid

To handle large-scale environments efficiently, a projected grid method is used. The mesh increases vertex density near the viewer while it reduces detail at greater distances. This ensures both visual fidelity and real-time performance.

Procedural wave texture generation

Wave motion is represented by using a procedural wave generation system that creates tileable, reusable wave patterns. These patterns are evaluated over a small domain and reused across the entire surface, enabling complex and continuous ocean motion at minimal computational cost.

Visual Enhancements

To further improve realism, several rendering enhancements were introduced. These include atmospheric effects, improved water shading, and support for both above water and underwater camera perspectives.
