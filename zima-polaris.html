Zima Polaris

Zima Polaris is a 2D platformer developed by a team of 14 people.

In this project, I worked as the lead engine programmer, developing core gameplay systems, engine architecture, and internal tools in C++ to support both gameplay and content creation.

Engine

I contributed to the development of a custom game engine tailored for 2D platforming mechanics. I worked on core systems including component-based architecture, integration of real-time physics and collision handling, animation state management, and input responsiveness.

Additionally, I implemented features such as resource management, scene serialization, and debugging tools.

Editor

I designed and implemented a custom in-engine level editor using Dear ImGui. The editor includes scene hierarchy management, object inspection, transformation tools, multi-object selection, and a full undo/redo system.

To improve iteration speed, I developed dedicated tools for level designers and artists, enabling efficient placement of geometry and scene prototyping. I worked closely with artists and designers to ensure the tools matched production needs, reducing friction in the workflow and significantly improving content creation efficiency.
