Deferred Rendering Engine

Deferred Rendering Engine

This project is a real-time deferred rendering pipeline focused on efficiently handling multiple dynamic light sources.

The system combines modern rendering techniques such as deferred shading, advanced shadow mapping, and screen-space post-processing to produce a visually rich and performant graphics pipeline.

Deferred Shading Pipeline

The renderer is built around a deferred shading architecture, separating geometry processing from lighting calculations. This allows the system to efficiently handle multiple lights without the traditional performance cost of forward rendering.

Light Volume Optimization

Point lights are optimized using a light volume approach, where lighting is computed only for fragments inside each light’s influence volume. This significantly reduces unnecessary shading computations and improves performance in complex scenes.

Cascaded Shadow Mapping

High-quality directional shadows are achieved through cascaded shadow maps, which divide the camera view into multiple depth ranges. This ensures sharp and stable shadow detail across both near and far view distances.

Decals and Surface Detail

A flexible decal system was implemented to project surface details such as damage, markings, and environmental effects directly onto geometry without modifying the underlying mesh.

Post-Processing Effects

To enhance visual quality, the pipeline integrates Alchemy Ambient Occlusion (AO) and bloom post-processing effects, improving depth perception and overall image richness.
