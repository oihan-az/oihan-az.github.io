Crowd Simulation

This project is a real-time crowd simulation system developed in C++ and Unreal Engine. It focuses on realistic agent navigation and interaction in dense environments.

My contribution implements collision avoidance using a probabilistic approach. Collision probability fields combined with density awareness dynamically correct agents’ preferred velocities, resulting in smoother motion, reduced oscillations, and more realistic crowd behavior in high-density scenarios.

Probability-based collision avoidance

Each agent dynamically adjusts its trajectory based on a probabilistic collision field computed from nearby agents and static obstacles. The system evaluates collision likelihood and local density to determine velocity corrections, ensuring smooth and realistic crowd navigation.

Density awareness

Agents adjust their preferred velocity by sampling multiple candidate velocities around the original preferred velocity. Each candidate is evaluated based on the local density of nearby agents, and the agent selects the velocity that best balances motion efficiency and collision avoidance in dense scenarios.
