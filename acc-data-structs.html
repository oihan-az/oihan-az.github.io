Acceleration Data Structures

This project implements multiple acceleration data structures for efficient scene rendering, ray intersection, and collision detection:

  • Uniform Grid: Subdivides space into uniform grids of the same size.
  • Bounding Volume Hierarchies (BVH): Hierarchical structures where objects are encapsulated in bounding volumes, and these volumes are recursively grouped into a hierarchy. Mainly used in static scenes.
  • Octree: Hierarchical subdivision for dynamic scenes, where each cell is represented using a location code as a hash. Works efficiently in dynamic environments.

These data structures combine computer graphics and physics techniques to optimize rendering and real-time interaction with both static and dynamic scenes.
